# SaaS Project Kit (Template)

Opinionated GitHub template for fast, safe SaaS delivery with 1–3 agents (REBAS + fury + builder).

Includes:
- PR/Issue/ADR/API templates
- CI gates: tests, lint, SAST (CodeQL), deps/secret/IaC scans
- ADR/spec consistency checks
- Preview build hooks

How to use
1) Click "Use this template" or clone and push to a new repo.
2) Run setup checklist in docs/SETUP.md.
3) Start with docs/PRD.md → docs/ux/ → docs/api/openapi.yaml → code.
